# Product Requirements Document: Exai MVP

## Product Overview

App Name: Exai Tagline: Practice smarter for exams with AI-generated
questions from your own materials. Launch Goal: Create an income source
reaching \$500 revenue Target Launch: 2 weeks

## Target Users

University students studying for quizzes, midterms, and finals.

## Core Problem

ChatGPT and similar tools can generate questions but the studying
experience is messy and unstructured. Students struggle to focus on one
question, track answers, and identify weak topics.

## Must-Have MVP Features

1.  Upload past exams (PDF / photo / text)
2.  AI generates similar exam-style questions
3.  One-question-at-a-time practice interface
4.  Answer box + solution
5.  Weak topic tracking
6.  Question bank history
7.  User answer history

## Version 2 (Not in MVP)

-   Sharing exams
-   Downloading question sets as PDFs

## Success Metric

\$500 revenue

## Design Vibe

Clean, minimal, fast, focused, modern

## Key Screens

Dashboard -- upload exams and access question banks\
Practice Screen -- one question with answer box\
Results Screen -- see solution and continue\
Weak Topics Screen -- see weak topics and practice them

## Constraints

Budget: Under \$50/month\
Timeline: 2 weeks\
Team: Solo founder

## MVP Definition of Done

-   All core features work end‑to‑end
-   Mobile responsive
-   Payment integration works
-   Tested with real students

Status: Ready for Technical Design
